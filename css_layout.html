Diseño y Layout

Modelo de Caja

El modelo de caja describe cómo se comportan los elementos en CSS. Cada elemento se considera una caja con márgenes, bordes, rellenos y contenido. Ejemplo:

div {
        margin: 20px;
        border: 2px solid black;
        padding: 15px;
        width: 300px;
    }

Esto define un div con márgenes de 20 píxeles, borde de 2 píxeles, relleno de 15 píxeles y un ancho de 300 píxeles.

Flexbox

Flexbox es un modelo de diseño que permite alinear y distribuir espacio entre elementos. Ejemplo:

.container {
        display: flex;
        justify-content: space-between;
    }

Esto configura un contenedor flex que distribuye el espacio entre sus elementos hijos.

Grid

Grid es un sistema de diseño bidimensional. Ejemplo:

.grid {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
    }

Esto crea un contenedor de cuadrícula con 3 columnas de igual ancho.

Posicionamiento

El posicionamiento en CSS permite controlar la ubicación de un elemento. Ejemplo:

div {
        position: absolute;
        top: 50px;
        left: 100px;
    }

Esto posiciona el div a 50 píxeles desde la parte superior y 100 píxeles desde la izquierda.

Media Queries

Las media queries permiten aplicar estilos CSS según el tamaño de la pantalla. Ejemplo:

@media (max-width: 600px) {
        body {
            background-color: lightblue;
        }
    }

Esto cambia el color de fondo a azul claro en pantallas de 600 píxeles o menos.
