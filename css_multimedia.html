Multimedia y Animaciones

Imágenes y Videos

Se pueden insertar imágenes y videos en una página web utilizando las etiquetas <img> y <video>. Ejemplo:

<img src="imagen.jpg" alt="Descripción de la imagen">
<video controls>
        <source src="video.mp4" type="video/mp4">
        Tu navegador no soporta el video.
    </video>

Animaciones en CSS

Las animaciones en CSS permiten crear efectos visuales al cambiar propiedades CSS. Ejemplo:

@keyframes ejemplo {
        from { opacity: 0; }
        to { opacity: 1; }
    }

    div {
        animation: ejemplo 2s ease-in-out;
    }

Transiciones

Las transiciones permiten cambiar suavemente una propiedad CSS de un estado a otro. Ejemplo:

div {
        transition: background-color 0.5s ease;
    }

    div:hover {
        background-color: yellow;
    }

Transformaciones

Las transformaciones permiten aplicar efectos como rotaciones. Ejemplo:

div {
        transform: rotate(45deg);
    }

Uso de Iframes

Los iframes permiten insertar contenido de otras páginas web dentro de un documento. Ejemplo:

<iframe src="https://www.ejemplo.com" width="300" height="200"></iframe>
