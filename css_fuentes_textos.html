Fuentes y Textos

Tipos de Fuente

CSS permite definir diferentes tipos de fuente mediante la propiedad font-family. Ejemplo:

h1 {
        font-family: 'Arial', sans-serif;
    }

Esto establece que el encabezado h1 utilizará la fuente Arial o, si no está disponible, una fuente sans-serif.

Propiedades de Texto

Las propiedades de texto incluyen font-size, font-weight, y line-height. Ejemplo:

p {
        font-size: 16px;
        font-weight: bold;
    }

Esto establece que los párrafos tendrán un tamaño de fuente de 16 píxeles y estarán en negrita.

Espaciado y Alineación

El espaciado se puede ajustar con margin y padding, y la alineación del texto se controla con text-align. Ejemplo:

h2 {
        margin-top: 20px;
        text-align: center;
    }

Esto establece un margen superior de 20 píxeles y centra el texto del encabezado h2.

Estilos de Texto

Se pueden aplicar estilos como negrita (font-weight: bold;), cursiva (font-style: italic;), y subrayado (text-decoration: underline;). Ejemplo:

em {
        font-style: italic;
    }

Esto aplicará un estilo en cursiva a todos los elementos <em>.

Uso de Google Fonts

Google Fonts ofrece una amplia variedad de fuentes que se pueden incluir en un proyecto utilizando un enlace en el <head> del documento. Ejemplo:

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
