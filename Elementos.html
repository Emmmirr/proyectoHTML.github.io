Etiquetas HTML
Ordenadas Alfabéticamente




Etiqueta Descripción
<!--...--> Define un comentario
<!DOCTYPE> Define el tipo de documento
<a> Define un hipervínculo
<abbr> Define una abreviatura o un acrónimo
<audio> Define contenido de sonido incrustado
<b> Define texto en negrita
<base> Especifica la URL/base de destino para todas las URLs relativas en un documento
<body> Define el cuerpo del documento
<br> Define un salto de línea
<button> Define un botón clicable
<div> Definir una sección en un documento
<dl> Definir una lista de descripción
<dt> Definir un término/nombre en una lista de descripción
<form> Definir un formulario HTML para la entrada del usuario
<h1> a <h6> Definir encabezados HTML
<head> Contiene metadatos/información para el documento
<img> Definir una imagen
<input> Defina un control de entrada
<label> Defina una etiqueta para un elemento <input>
