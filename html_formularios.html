Formularios

Estructura de un Formulario

Un formulario se define utilizando la etiqueta <form>. Este elemento permite recoger datos del usuario que se pueden enviar a un servidor.

<form action="ruta_a_la_accion" method="post">

    <!-- Campos del formulario -->

    </form>

Campos de Entrada

Los campos de entrada permiten al usuario ingresar datos. Algunos de los tipos más comunes son:

  • <input type="text">: Campo de texto simple.
  • <input type="password">: Campo de entrada de contraseña, donde los caracteres son ocultados.
  • <textarea>: Área de texto que permite la entrada de múltiples líneas.
<input type="text" name="nombre">

<input type="password" name="contrasena">

<textarea name="comentarios"></textarea>

Botones

Los botones en un formulario permiten enviar o reiniciar datos. Los tipos más comunes son:

  • <input type="submit">: Envía el formulario.
  • <input type="reset">: Reinicia los campos del formulario a sus valores por defecto.
  • <button>: Permite crear botones personalizados con contenido HTML.
<input type="submit" value="Enviar">

<input type="reset" value="Reiniciar">

<button type="button">Botón Personalizado</button>

Listas Desplegables y Casillas de Verificación

Estos elementos permiten seleccionar opciones:

  • <select>: Crea una lista desplegable para seleccionar una opción.
  • <input type="checkbox">: Crea casillas de verificación que permiten seleccionar múltiples opciones.
<select name="opcion">

    <option value="opcion1">Opción 1</option>

    <option value="opcion2">Opción 2</option>

</select>

<input type="checkbox" name="opcion1"> Opción 1

<input type="checkbox" name="opcion2"> Opción 2

Validación de Formularios

La validación de formularios se utiliza para verificar los datos ingresados antes de enviarlos. Esto se puede hacer utilizando atributos HTML como required y pattern, o mediante JavaScript para una validación más avanzada.

<input type="text" name="email" required>

<input type="text" name="telefono" pattern="[0-9]{10}" title="Debe tener 10 dígitos">
