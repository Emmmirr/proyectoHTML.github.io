Introducción a CSS

Definición: CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Permite aplicar estilos a los elementos de una página web.

Propósito: CSS se utiliza para separar el contenido de la presentación, lo que permite un diseño más flexible y un mantenimiento más sencillo del código.

¿Qué es CSS?

CSS es el lenguaje que define cómo se mostrarán los elementos HTML en la pantalla. Incluye propiedades para colores, fuentes, diseño y mucho más.

Selectores en CSS

Los selectores son patrones que se utilizan para seleccionar los elementos que se van a estilizar. Ejemplo:

p { color: blue; }

Este ejemplo cambiará el color de todos los párrafos a azul.

Propiedades y Valores

Las propiedades CSS definen qué aspecto tendrá un elemento, y los valores son las especificaciones de esas propiedades. Ejemplo:

h1 { font-size: 24px; }

Este código establece el tamaño de la fuente del encabezado h1 a 24 píxeles.

Modelos de Caja

El modelo de caja describe cómo se comportan los elementos en CSS. Cada elemento se considera una caja con márgenes, bordes, rellenos y contenido.

div {
        margin: 10px;
        border: 1px solid black;
        padding: 5px;
    }

Este ejemplo define un div con márgenes de 10 píxeles, un borde de 1 píxel y relleno de 5 píxeles.

Comentarios en CSS

Los comentarios en CSS se utilizan para hacer anotaciones en el código. Se escriben así:

/* Este es un comentario */
