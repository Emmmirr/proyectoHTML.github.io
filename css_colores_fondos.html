Colores y Fondos

Colores en CSS

Los colores en CSS se pueden definir utilizando nombres de color, valores hexadecimales, o valores RGB y RGBA. Ejemplo:

body { background-color: #f0f0f0; }

Este código establece el color de fondo del cuerpo de la página a un gris claro.

Propiedades de Fondo

Las propiedades de fondo permiten definir el color de fondo, imágenes de fondo y otros estilos de fondo. Ejemplo:

div {
        background-color: blue;
        color: white;
    }

Esto establece un fondo azul y texto blanco dentro del div.

Gradientes

Los gradientes permiten crear transiciones suaves entre dos o más colores. Ejemplo:

div {
        background: linear-gradient(to right, red, blue);
    }

Esto aplica un gradiente lineal que va de rojo a azul de izquierda a derecha.

Imágenes de Fondo

Las imágenes de fondo se definen con la propiedad background-image. Ejemplo:

body {
        background-image: url('imagen.jpg');
    }

Esto establece una imagen de fondo para el cuerpo de la página.

Transparencias

La transparencia se puede manejar con el valor alpha en los colores RGBA. Ejemplo:

div {
        background-color: rgba(255, 0, 0, 0.5); /* rojo con 50% de opacidad */
    }
