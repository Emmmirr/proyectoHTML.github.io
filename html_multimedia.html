Multimedia en HTML

Inserción de Imágenes

Las imágenes se insertan en una página HTML utilizando la etiqueta <img>. Se deben especificar los atributos src (fuente) y alt (texto alternativo).

<img src="ruta_de_la_imagen.jpg" alt="Descripción de la imagen">

Uso de Vídeos

Los vídeos se pueden incluir usando la etiqueta <video>. Es importante incluir atributos como controls para que el usuario pueda reproducir el vídeo.

<video controls>

    <source src="ruta_del_video.mp4" type="video/mp4">

    Tu navegador no soporta la etiqueta de video.

    </video>

Audio

El audio se inserta utilizando la etiqueta <audio>. Al igual que con el vídeo, el atributo controls permite a los usuarios controlar la reproducción.

<audio controls>

    <source src="ruta_del_audio.mp3" type="audio/mpeg">

    Tu navegador no soporta la etiqueta de audio.

    </audio>

Integración de Iframes

Los iframes permiten incrustar otra página web dentro de la página actual. Se utiliza la etiqueta <iframe>.

<iframe src="url_de_la_pagina" width="600" height="400"></iframe>

Optimización de Multimedia

Para mejorar el rendimiento de las imágenes y vídeos, se recomienda:

  • Utilizar formatos adecuados (como WebP para imágenes).
  • Comprimir imágenes antes de subirlas.
  • Usar atributos width y height para establecer dimensiones.
