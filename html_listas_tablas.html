Listas y Tablas

Listas no Ordenadas

Las listas no ordenadas se utilizan para crear listas con viñetas. Se define utilizando la etiqueta <ul> para la lista y <li> para cada elemento.

<ul>

    <li>Elemento 1</li>

    <li>Elemento 2</li>

    <li>Elemento 3</li>

</ul>

Listas Ordenadas

Las listas ordenadas se utilizan para crear listas numeradas. Se define utilizando la etiqueta <ol> para la lista y <li> para cada elemento.

<ol>

    <li>Primer elemento</li>

    <li>Segundo elemento</li>

    <li>Tercer elemento</li>

</ol>

Listas de Definición

Las listas de definición se utilizan para definir términos. Se define utilizando la etiqueta <dl> para la lista, <dt> para el término y <dd> para la definición.

<dl>

    <dt>Término 1</dt>

    <dd>Definición del término 1.</dd>

    <dt>Término 2</dt>

    <dd>Definición del término 2.</dd>

</dl>

Tablas

Las tablas se utilizan para organizar datos en filas y columnas. Se define utilizando la etiqueta <table>, con <tr> para las filas, <th> para los encabezados de columna y <td> para las celdas de datos.

<table>

    <tr>

        <th>Encabezado 1</th>

        <th>Encabezado 2</th>

    </tr>

    <tr>

        <td>Dato 1</td>

        <td>Dato 2</td>

    </tr>

</table>

Fusionar y Dividir Celdas

Para fusionar celdas en una tabla, se utilizan los atributos colspan y rowspan. El primero se utiliza para fusionar celdas horizontalmente, y el segundo para fusionarlas verticalmente.

<table>

    <tr>

        <td colspan="2">Celda fusionada</td>

    </tr>

    <tr>

        <td>Dato 1</td>

        <td rowspan="2">Celda dividida</td>

    </tr>

    <tr>

        <td>Dato 2</td>

    </tr>

</table>

Formateado Avanzado de Tablas

Para un formateo avanzado de tablas, se pueden aplicar estilos CSS a las tablas, filas, celdas y encabezados. Esto permite personalizar el diseño de las tablas, como el color de fondo, bordes y espaciado.

table {

    border-collapse: collapse;

}

th, td {

    border: 1px solid black;

    padding: 8px;

    text-align: left;

}
