¿Qué es HTML?

Definición: HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Sirve como la estructura fundamental de cualquier sitio web, definiendo la organización y presentación del contenido.

Propósito: HTML utiliza etiquetas (tags) para describir la estructura de un documento, como títulos, párrafos, imágenes, enlaces y más. Los navegadores web interpretan estas etiquetas y muestran el contenido de forma visual en la pantalla.

Estructura básica de un documento HTML

<html>: Esta etiqueta envuelve todo el contenido de una página HTML.

<head>: Contiene información meta sobre la página, como el título, estilos CSS y scripts.

<body>: Aquí se coloca el contenido visible de la página, es decir, todo lo que el usuario verá en el navegador.

Elementos de Texto

Encabezados

Se utilizan para definir títulos y subtítulos. Las etiquetas van desde <h1> (el más grande) hasta <h6> (el más pequeño).

Párrafos

Se utilizan para organizar el texto en bloques. La etiqueta es <p>.

Saltos de Línea

La etiqueta <br> fuerza un salto de línea en un punto específico.

Formato de Texto

HTML ofrece etiquetas para enfatizar texto, como <b> y <i>, y marcar texto como importante, como <strong> y <em>.

Comentarios en HTML

Los comentarios se utilizan para agregar notas explicativas al código sin que sean mostrados en el navegador. Se escriben entre <!-- comentario -->.

DocType

El <!DOCTYPE html> al inicio del documento especifica el tipo de documento HTML que se está utilizando. Es importante para que el navegador interprete correctamente el código.
