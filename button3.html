Definición y uso

La <button> etiqueta define un botón en el que se puede hacer clic.

Dentro de un <button> elemento puedes poner texto (y etiquetas como <i>, <b>, <strong>, <br>, <img>, etc.). ¡Eso no es posible con un botón creado con el <input>elemento!

Consejo: especifique siempre el typeatributo de un <button> elemento para indicar a los navegadores qué tipo de botón es.
