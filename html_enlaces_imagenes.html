Enlaces y Imágenes

Hipervínculos

Los hipervínculos son elementos en una página web que permiten navegar a otra página o a una sección específica dentro de la misma. Se pueden crear enlaces internos (a otras páginas dentro del mismo sitio) y externos (a páginas en otros sitios).

<a href="URL">Texto del enlace</a>

Imágenes

Para insertar una imagen en HTML, se utiliza la etiqueta <img>. Los atributos principales son:

  • src: Especifica la ruta de la imagen.
  • alt: Proporciona un texto alternativo para la imagen, importante para la accesibilidad.
  • width: Define el ancho de la imagen (en píxeles o porcentajes).
  • height: Define la altura de la imagen (en píxeles o porcentajes).
<img src="ruta_de_la_imagen" alt="Descripción de la imagen" width="200" height="150">

Mapas de Imagen

Los mapas de imagen permiten crear áreas interactivas dentro de una imagen. Se utilizan las etiquetas <map> y <area> para definir las zonas activas y sus enlaces.

<img src="imagen.jpg" usemap="#mapa">

<map name="mapa">

    <area shape="rect" coords="34,44,270,350" href="enlace.html" alt="Descripción">

</map>

Imágenes como Enlaces

Las imágenes también se pueden utilizar como enlaces. Para ello, se coloca la etiqueta <img> dentro de un hipervínculo.

<a href="enlace.html"><img src="imagen.jpg" alt="Descripción"></a>

Optimización de Imágenes

Optimizar imágenes es crucial para mejorar el rendimiento de una página web. Aquí algunos consejos:

  • Usar formatos de imagen adecuados (JPEG, PNG, WebP).
  • Comprimir imágenes para reducir su tamaño sin perder calidad.
  • Especificar dimensiones de imágenes para mejorar el tiempo de carga.
  • Utilizar atributos alt para mejorar la accesibilidad y SEO.
