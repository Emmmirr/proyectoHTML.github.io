CSS Avanzado

Preprocesadores CSS

Los preprocesadores como SASS y LESS permiten utilizar características avanzadas en CSS. Ejemplo en SASS:

$color-primario: #3498db;

    body {
        background-color: $color-primario;
    }

Variables en CSS

Las variables CSS permiten almacenar valores que se pueden reutilizar en el código. Ejemplo:

:root {
        --color-primario: #3498db;
    }

    body {
        background-color: var(--color-primario);
    }

Funciones en CSS

CSS permite usar funciones como calc() para realizar cálculos. Ejemplo:

div {
        width: calc(100% - 20px);
    }

Filtros CSS

Los filtros permiten aplicar efectos visuales a imágenes y elementos. Ejemplo:

img {
        filter: grayscale(100%);
    }

Clip-path

La propiedad clip-path permite crear formas personalizadas. Ejemplo:

div {
        clip-path: circle(50%);
    }
