Semántica en HTML

Importancia de la Semántica

La semántica en HTML se refiere al uso adecuado de las etiquetas para dar significado al contenido. Mejora la accesibilidad y el SEO (optimización para motores de búsqueda).

Etiquetas Semánticas

Algunas de las etiquetas semánticas más utilizadas son:

  • <header>: Define un encabezado para un documento o sección.
  • <nav>: Sección para enlaces de navegación.
  • <article>: Contenido independiente y autocontenido.
  • <section>: Sección temática dentro de un documento.
  • <footer>: Información de pie de página para un documento o sección.
<header>

    <h1>Título Principal</h1>

    </header>

    <nav>

    <ul>

    <li><a href="#>Enlace 1</a></li>

    </ul>

    </nav>

Uso de <figure> y <figcaption>

Las etiquetas <figure> y <figcaption> se utilizan para encapsular imágenes y sus descripciones.

<figure>

    <img src="imagen.jpg" alt="Descripción de la imagen">

    <figcaption>Esta es una descripción de la imagen.</figcaption>

    </figure>

Ventajas de Usar Etiquetas Semánticas

Usar etiquetas semánticas ofrece varios beneficios:

  • Mejora la accesibilidad para personas con discapacidad.
  • Facilita la indexación y mejora el SEO.
  • Proporciona una estructura más clara y legible para el código.
